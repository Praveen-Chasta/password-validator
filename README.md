# 🔐 Password Strength Validator

A simple React app that validates the strength of a password based on common security rules such as length, uppercase, lowercase, number, and special character.

## 🚀 Features

- Real-time password validation
- Displays dynamic message (Strong / Not a Strong Password)
- Conditional message styling (green/red)
- Easy to use input field

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Styling**: Plain CSS
